# University Study Program Landing Page

Base scaffold for a university study program landing page.

## Structure

- `index.html` - main entry point
- `css/input.css` - custom stylesheet entry
- `js/app.js` - client-side behavior
- `js/data.js` - data placeholders for Alpine.js
- `assets/` - static media folders

## Prototype Stack

- HTML
- Tailwind CSS via CDN
- Alpine.js via CDN
